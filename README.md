# 教育数字化课题研究技能包

> 基于《生成式人工智能赋能教育数字化课题研究指导文档》开发的 OpenCode 技能套件

## 概述

本技能包专注于**教育数字化课题研究**场景，为中小学教师、高校研究人员、区域教育研究者提供从**选题申报到成果总结**的全流程支持。

## 核心价值

| 价值维度 | 说明 |
|---------|------|
| 提升教学质量 | 通过科研更新教育理念与方法 |
| 促进教师发展 | 提升数字技术应用能力与科研素养 |
| 推动教育改革 | 为教育数字化政策落地提供实践依据 |
| 强化团队协作 | 搭建跨学科研究共同体 |

## 技能清单

### EDU-AI-001 课题选题生成

- **功能**：基于研究方向、实践场景与政策导向，生成符合规范的课题名称
- **输入**：研究方向、实践领域、字数要求
- **输出**：3-5个候选题目及优化建议
- **场景**：课题申报初期选题

### EDU-AI-002 申报书框架生成

- **功能**：生成课题申报书各模块框架，提供撰写指导与示例
- **输入**：课题名称、研究类型
- **输出**：完整申报书框架、各模块撰写要点
- **场景**：课题申报书撰写

### EDU-AI-003 文献解读与研究现状分析

- **功能**：解析学术文献核心内容，梳理研究进展与不足
- **输入**：文献全文/摘要、研究主题
- **输出**：核心观点、研究现状分析报告、检索建议
- **场景**：研究现状述评撰写

### EDU-AI-004 调研工具生成

- **功能**：基于研究变量与维度，生成问卷、访谈提纲等调研工具
- **输入**：研究主题、核心变量、目标群体
- **输出**：规范调研工具（含量表、指导语）
- **场景**：数据收集阶段

### EDU-AI-005 研究过程优化建议

- **功能**：基于研究进展与反馈，提供方法调整、流程优化建议
- **输入**：当前研究阶段、遇到的问题、现有方案
- **输出**：问题分析、优化策略、实施步骤
- **场景**：课题实施阶段

### EDU-AI-006 研究成果撰写与润色

- **功能**：生成研究报告、论文等成果初稿，优化语言表达与逻辑结构
- **输入**：研究框架、核心数据、成果类型
- **输出**：成果初稿、润色版本、修改建议
- **场景**：成果总结阶段

### EDU-AI-007 GAI应用伦理审核

- **功能**：检查AI生成内容的合规性，识别学术诚信与数据安全风险
- **输入**：AI生成内容、应用场景
- **输出**：合规性审核报告、修改建议
- **场景**：全研究周期

## 选题方向参考

- 人工智能教育平台与空间建设研究
- 人工智能教育资源开发与应用研究
- 教师数字素养提升与队伍建设研究
- 人工智能赋能教学模式与机制研究
- 区域教育数字化协同发展研究

## 选题原则

1. **具体明确**：清晰体现理论基础、研究方法、研究对象与核心内容
2. **问题导向**：聚焦教学实践中的真实困境
3. **政策契合**：紧扣教育数字化、核心素养等国家政策导向
4. **可行性强**：结合研究团队能力与资源条件

## 课题研究标准流程

```
选题 → 申报书撰写 → 文献研究 → 数据收集 → 实施推进
  → 成果撰写 → 伦理审核 → 总结结题
```

## 使用示例

### 示例1：课题选题

**用户输入：**
> 我是教育技术学教师，关注"人工智能赋能高等教育"方向，负责师范生教学，帮我生成课题名称

**技能输出：**
1. 《生成式AI赋能师范生教学能力提升策略研究》
2. 《人工智能支持的高校师范生培养模式创新研究》
3. 《师范生数字素养提升的人机协同教学路径研究》
4. 《AI赋能师范生课堂教学设计能力发展研究》
5. 《生成式人工智能助力师范生专业成长路径研究》

### 示例2：申报书框架

**用户输入：**
> 帮我生成课题申报书框架，课题名称是《生成式AI赋能小学语文个性化教学策略研究》

**技能输出：**
- 选题背景与价值
- 国内外研究现状述评
- 研究目标与内容
- 研究思路、方法与步骤
- 可行性分析
- 预期研究成果

将 `edu-ai-skills.zip` 解压到 OpenCode skills 目录：


## GAI应用伦理规范

使用本技能包时，请遵守以下伦理规范：

1. **学术诚信**：明确AI工具的使用范围，注明AI生成内容的来源
2. **数据安全**：保护研究过程中收集的个人信息，确保数据合规使用
3. **内容审核**：对AI生成的内容进行人工审核，纠正"幻觉"与错误信息
4. **原创性要求**：以AI为辅助工具，融入研究团队的原创思考与实践经验

## 注意事项

- 生成的题目需符合学术规范，避免过大过空
- 最终选题需结合团队实际能力确定
- 涉及伦理合规类内容，必须经人工审核确认后方可使用

## 附录：常见问题

| 常见问题 | 解决策略 |
|----------|----------|
| 选题过大过空 | 采用"小切口"选题思路，聚焦具体学段、学科 |
| 研究方法与内容不匹配 | 重新梳理研究内容核心需求，选择适配方法 |
| 过程性资料收集不足 | 建立资料管理清单，定期整理归档 |
| 技术与教学融合浅层化 | 深入分析学科教学特点，设计针对性应用场景 |
| 研究逻辑闭环缺失 | 强化"问题-目标-内容-成果"对应关系 |
| AI工具应用不规范 | 制定AI使用指南，明确使用场景与审核流程 |
